== Description ==

Once activated the plugin will start to display falling snowflakes over the pages of your WordPress blog website.

The plugin performs this by including a JavaScript code snippet before the closing body tag of each page.

== Installation ==

1. Upload `floating-snowflakes.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

= Do I have to specify or upload a snowflake image ? =

No, the plugin actually uses * (star) characters with different sizes as snowflakes.
